CandyUI 消息提示样式展示

共30种独特的消息提示样式，支持3D效果和动画

使用方法

点击按钮查看不同类型的消息提示效果。提示会在3秒后自动消失。

// 显示成功消息 CandyToast.success('操作成功！');
// 显示错误消息 CandyToast.error('发生错误！');
// 显示警告消息 CandyToast.warning('请注意！');
// 显示信息消息 CandyToast.info('这是一条信息提示。');
// 显示加载消息 CandyToast.loading('正在加载...');

API 示例

JavaScript API

// 基本用法 CandyToast.show('消息内容');
// 带配置 CandyToast.show('消息内容', { type: 'success', style: 1, duration: 5000, position: 'top-right' });
// 快捷方法 CandyToast.success('成功消息'); CandyToast.error('错误消息'); CandyToast.warning('警告消息'); CandyToast.info('信息消息'); CandyToast.loading('加载中...');
// 全局配置 CandyToast.config({ position: 'top-right', duration: 3000, maxCount: 5 });
// 关闭所有提示 CandyToast.dismissAll();
// 关闭指定提示 const id = CandyToast.success('提示'); CandyToast.dismiss(id);
