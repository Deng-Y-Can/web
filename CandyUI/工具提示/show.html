CandyUI 工具提示样式展示

共30种独特的工具提示样式，支持3D效果和动画

使用方法

将鼠标悬停在按钮上查看效果。工具提示会自动显示和隐藏。

<button class="trigger-btn" data-tooltip="提示内容">悬停查看</button>
CandyTooltip.bind(element, '提示内容', { position: 'top' });

API 示例

JavaScript API

// 显示提示 CandyTooltip.show(element);
// 隐藏提示 CandyTooltip.hide(element);
// 切换提示 CandyTooltip.toggle(element, '新内容');
// 创建提示 CandyTooltip.create({ content: '提示内容', position: 'top', theme: 'default', delay: 0, duration: 3000 });
// 全局配置 CandyTooltip.config({ position: 'top', delay: 200, duration: 3000 });
